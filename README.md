
# Roadwave 

Online shop website that sells fashion necessities such as t-shirts, shirts, trousers and outwear. This project is the final assignment of the Web Design and Programming course.


## Features

- Login and Account Registration
- Dual Role (Client and Admin)
- Dark Mode
- Admin can carry out product management (CRUD)

## Missing Features

- Add Product to Cart
- Add Product to Wishlist
- Checkout Product
- Payment Gateway



## Tech Stack

PHP, Laravel, TailwindCSS, MySQL
